## ğŸ¨ Customization Guide

### Update Personal Information

1. **Profile Photo**: Replace `ADESH-HR.jpg` with your photo
2. **Contact Details**: Update email and phone in the footer section
3. **Resume Link**: Update the Google Drive link in the "Download Resume" button
4. **Projects**: Modify the project cards in the `#projects` section

### Color Scheme

The portfolio uses a custom color palette defined in Tailwind config:

```javascript
colors: {
  'background-dark': '#121212',
  'card-dark': '#1e1e1e',
  'accent-teal': '#00bcd4',
  'text-light': '#f0f0f0',
}
```

Modify these values in the `<script>` tag to change the entire theme!
